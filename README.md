# Smart Expense Tracker

A Python-based command-line application that helps students track daily expenses and analyze spending habits.

## Features

- Add Expenses
- View Expense History
- Category-wise Summary
- CSV Storage
- Error Handling

## Technologies

- Python
- CSV
- Git
- GitHub
